## Goals

- Learn fundamentals and advanced concepts across multiple technologies.
- Build practical projects for each language/area.
- Track progress with notes, exercises, and examples.
- Keep everything consistent and easy to navigate.

## How to Use This Repo

1. Pick a topic folder.
2. Add your project, notes, or exercises.
3. Use clear file and folder names.
4. Commit regularly to track progress.

## Naming Conventions

- Use lowercase folder names.
- Use hyphens for multi-word names.
- Keep project names short and descriptive.

Examples:

- `web-dev/portfolio-site`
- `python/data-cleaner`
- `java/banking-app`
- `cpp/graph-algorithms`

## Contribution

If you are collaborating with others:

1. Create a branch for your work.
2. Open a pull request with a clear description.
3. Keep changes focused on one topic or feature.
