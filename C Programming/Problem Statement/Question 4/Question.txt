Implement a Client–Server program in C using socket programming for inter-process communication (IPC). The server should accept arithmetic operation requests from the client in the format <OP> <A> <B>, where <OP> ∈ {ADD, SUB, MUL, DIV, MOD}. The server should parse the request, perform the computation, and send the result back to the client. The client should accept input from the user, forward it to the server, and display the response. Demonstrate the program with at least three valid inputs and one error case (e.g., division by zero or invalid operator).
